# Kali Custom Launchers

Desktop `.desktop` launchers and companion scripts for non-Kali-native apps on this machine — Wi-Fi, BLE, SDR, GSM, Tor, Grok, and utilities.

Collected automatically by `gh-launcher-sync` from:

- `~/Desktop/*.desktop`
- `~/.local/share/applications/*.desktop`
- Referenced scripts under `~/bin` and `~/.local/bin`

## App catalog

See **[APPS.md](APPS.md)** for a full list of every launcher and what it does.

## Layout

| Path | Contents |
|---|---|
| `desktop/` | Launcher `.desktop` files |
| `bin/` | Scripts and launcher subdirectories |
| `icons/` | Custom icons referenced by launchers |
| `MANIFEST.txt` | Collection log (auto-generated) |
| `APPS.md` | Human-readable app catalog |

## Sync

Run **GitHub Launcher Sync** from the desktop, or:

```bash
$HOME/bin/gh-launcher-sync/gh-launcher-sync.sh
# or quick push:
$HOME/bin/gh-launcher-sync/push-now.sh
```
